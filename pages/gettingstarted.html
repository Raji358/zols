Prerequisites

  1. JDK 1.7
  2. Maven
  3. MongoDB
  4. Git

Steps

Start a Mongo DB server as documented in the Mongo DB website

  • git clone https://github.com/sathishk/zols.git
  • cd zols
  • mvn clean install
  • cd ..
  • git clone https://github.com/sathishk/zols-sample.git
  • cd zols-sample
  • mvn tomcat7:run

You can access the application @ http://localhost:8080/zols-sample Login to the application using username/password as admin/admin.
