Prerequisites

  1. JDK 1.7
  2. Maven
  3. MongoDB
  4. Git

Steps

  1. Start a mongo server as documented in the mongo website
  2. git clone https://github.com/sathishk/zols.git
  3. cd zols
  4. mvn clean install
  5. cd..
  6. git clone https://github.com/sathishk/zols-sample.git
  7. mvn tomcat7:run
You can access the application @ http://localhost:8080/zols-sample Login to the application using username/password as admin/admin.
