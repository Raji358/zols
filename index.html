Get Started

Embedded CMS

Zols can run as a standalone CMS or as an embedded CMS within an existing Spring Web Application. Include the necessary jars to your Spring Application, and CMS is enabled rightaway.

Direct HTML Linking

Directly link HTML files to CMS. Make changes locally and see it reflects strightaway in the site. This provides the ability to quickly bootstrap and test the HTML authoring, and deploy it to production

Native Data

Data gets stored and retrieved in a structured, native format(Dates, Time, Number etc) which can be accessed and modified by clearly documented REST API.
