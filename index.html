Get Started

Embedded CMS

Zols can run both as a standalone CMS or as an embedded CMS within your existing Spring Web Application. All you need is to include some jars to your Spring MVC application. Your CMS is in your application straight away!

Direct HTML Linking

Directly link HTML files to CMS. Make changes and see it reflects strightaway in the site. This gives the power of quickly bootstrapping and test HTML authoring and be able to quickly deploy it in the production.

Native Data

Data is stored in a structured, native format(E.g Dates, Time, Number etc). Which can be read, update, delete and query by clearly documented REST API
